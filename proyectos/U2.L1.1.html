¿Cómo se crea una carpeta?

Dando clic derecho luego te aparecera nuevo y te saldra la carpeeta

¿Cómo se abre una carpeta en VS?

Toma tu carpeta y la arrastra hasta la aplicacion VS

¿Qué es HTML?

Es un lenguaje para dar instrucciones a una computadora sobre como hacer una pagina

¿Cómo se crea un archivo de HTML en VS?

Dando clic en una carpetica que tiene un signo de mas

¿Qué etiquetas de HTML ya conocemos? Define cada etiqueta.

  1. HTML:Es el elemento raiz de una pagina HTML
  2. HEAD:Contiene informacion sobre la pagina HTML
  3. TITLE:Especifica un titulo para la pagina HTML
  4. BODY:El cuerpo del documento
  5. H1-H6:Encabezados
  6. P:Para parrafos
  7. EM:Pone el texto en cursiva
  8. STRONG:Pone el texto en negrita
  9. DIV:Es un contenedor que divide la pagina en secciones
  10. BR:Salta a otra linea
  11. DOCTYPE html:Define que es un documento html
  12. BUTTON:Texto de boton
  13. SPAN:Subraya
  14. UL:Esta es como una viñeta
  15. OL:ES una viñeta pero por orden
  16. LI:Para separar
  17. IMG:Para imagenes
